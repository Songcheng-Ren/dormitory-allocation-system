热帖

如何快速融入集体？

关灯聊天，聊聊风俗和地方特色。一般来说啊，新生入学彼此都是陌生人，可以先询问要不要一起去买生活用品啊，或者吃饭啊，逛校园之内的。这样增进了解。晚上睡觉时，关灯后聊天就是最好的。因为这个时候比较没有限制，容易说更多。最后吧，上课一起去，平常帮忙带点东西，提醒一些事，总的来说，关系会更加亲近一点。
　　　　　　　　　　　--来自大二学长

如何快速融入集体？

多找找共同的兴趣爱好，这样一来也有的话聊。买了零食分享给室友，这样会消除彼此之间的陌生感。多和室友一起出去，不论是做什么，这样才能最快的和室友们熟络起来。
　　　　　　　　　　　--来自大三学姐

良好宿舍关系的一些小提示：

—— 一起做些事情以了解彼此
—— 愿意聆听和说出自己的心声
—— 试着理解而不是急着评判和评价
—— 对不同的生活方式和价值观持开放的态度
—— 愿意做出妥协
—— 不要揣测和猜忌，而是努力了解事实情况
—— 说出你的预期
